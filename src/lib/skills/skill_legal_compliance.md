# SKILL CARD: Legal, Contracts & Compliance
**Load this when:** Reviewing agreements, rates, deliverables, or disclosure requirements.

---

## FTC Disclosure Rules
Every paid partnership must include a clear disclosure. Flag any content that is missing one.

| Required | Examples |
|---|---|
| Hashtags | `#ad`, `#sponsored`, `#partner`, `#paidpartnership` |
| Verbal (video) | Creator must say it out loud, not just in caption |
| Placement | Must be visible BEFORE the viewer clicks "more" — not buried |

> ⚠️ If disclosure is missing or buried, flag as a **compliance issue**.

---

## Contract Essentials
> Look for these in: Contracts tab → `Agreement`, `Signed`, `Terms`, `Exclusivity`

A valid influencer contract should include:
- **Deliverables** — exact content type, quantity, and platform
- **Go-live dates** — specific posting deadlines
- **Creator rate** — agreed fee and payment schedule
- **Usage rights** — can the brand repost or run paid ads using the content?
- **Exclusivity** — is the creator restricted from working with competitors?
- **Revision rounds** — how many edits the brand can request
- **Kill fee** — what happens if the campaign is cancelled

If any of these are missing from the Contracts tab, flag as **incomplete agreement**.

---

## SOW (Scope of Work) Review
> Look for these in: SOW tab → `Offer`, `Negotiated Rate`, `Rate`, `Demographics`, `Deal`

When reviewing the SOW tab, check for:
- Initial offer vs. final negotiated rate — was there a significant difference?
- Creator demographics — do they match the brand's target audience?
- Deliverable count and format agreed upon
- Any special terms (gifting, affiliate commission, exclusivity add-ons)

---

## Usage Rights Guide
| Rights Type | Meaning |
|---|---|
| Organic only | Brand can repost on owned channels only |
| Paid amplification | Brand can run ads using creator's content |
| Whitelisting | Brand runs ads directly from creator's account handle |
| Buyout | Brand owns the content fully for unlimited use |

> Higher usage rights = higher creator fee. Flag if usage rights are broad but rate seems low.

---

## Red Flags to Flag Immediately
- Contract not signed before content goes live
- No FTC disclosure on published content
- Creator posted for a competitor within exclusivity window
- Usage rights used beyond what was agreed
- Payment not matching the rate in the SOW tab
